# SmartCane-Back-end
SmartCane-Back-end
### 4학년 1학기 졸업작품 시각장애인용 딥러닝 기반 스마트 지팡이
#### 안드로이드 APP에서 딥러닝 호출을 위한 API 개발

💻서버 실행 방법
1. git clone
2. SmartCane 진입
3. 가상환경 생성
 > python -m venv <가상환경 이름>
4. 가상 환경 실행
 > source 가상환경/bin/activate
5. 패키지 다운로드
 > pip install -r requirements.txt
6. SmartCane 진입
7. manage.py 파일 경로에서 명령어 실행
 > python3 manage.py runserver
